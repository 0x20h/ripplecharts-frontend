Live Trade Feed

The live trade feed shows the most recent trades for the selected currency pair, the latest trade price and the 24 hour high, low, and volume.

The large number at the top is the last trade price. It is expressed as a currency pair. The number represents how much of the second currency (counter currency) you can get for one unit of the first currency (base currency).

The table below this is a live updating ticker of the latest trades. The colored bar shows if the trade price was higher, lower or the same as the previous trade. Left column: amount traded. Right column: the trade price.

Live Market Data

The dropdowns allow you to select a base currency and a counter currency. Select the currency first, and the issuer dropdown will update to show which issuers are available for the selected currency. On the right side, the current ledger number is represented, followed by a circle which indicates whether or not Ripple Charts is currently connected to the Ripple Network real time feed.

Connected to the Ripple Network
Disconnected from the Ripple Network

This chart below shows price change over time. Grey bars in the background represent trade volume for that time period. In the top left, select time intervals between each data point. The displayed time range will automatically adjust based on which interval you select. At the top right you can select between line or candlestick display.

Hover your mouse over the chart to see open, high, low, and close values for each of the data points.

Order Book

The current order book for the selected currency pair is below - these are orders that have been placed but not filled yet. Bids are orders to buy the base currency, and asks are orders to sell the base currency. The “total” column shows the total amount of currency you could get at that price. The blue line shows the depth of the order book, and the vertical line represents the midpoint between the best bid and ask price.
